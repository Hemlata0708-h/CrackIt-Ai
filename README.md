# CrackIt-Ai 🚀

CrackIt-Ai is a **full-stack, fully functional web application** designed to help users prepare for interviews with AI-powered assistance, real-time interaction, and integrated payment features.

---

## 🌟 Features

* 🔐 User Authentication (Login / Signup)
* 🤖 AI-powered interview assistance
* 💳 Payment Integration using Razorpay
* 📊 Structured interview workflow
* ⚡ Fast and responsive UI
* 🛠 Fully functional backend with REST APIs

---

## 🛠 Tech Stack

### Frontend

* React.js
* Vite
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### Services & Integrations

* Razorpay (Payment Gateway)
* OpenRouter / AI Services

### Tools

* Git & GitHub
* REST APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/Hemlata0708-h/CrackIt-Ai.git
cd CrackIt-Ai

### 2️⃣ Install dependencies

#### Client

cd client
npm install

#### Server

cd server
npm install

---

### 3️⃣ Run the project

#### Start Backend

cd server
npm start

#### Start Frontend

cd client
npm run dev

---

## 🔐 Environment Variables

Create a `.env` file inside the server folder and add:

PORT=your_port
MONGO_URI=your_database_url
RAZORPAY_KEY=your_key

⚠️ Do not share your `.env` file publicly.
